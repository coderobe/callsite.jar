# callsite.jar 

find callsites in a jar

Usage: `java -jar callsite.jar <target> <classpath> <method>`  
Example: `java -jar callsite.jar /path/to/my.jar java/io/PrintStream println`  


## Building

`gradle jar`


## Dependencies

asm 2.2.3  
asm-commons 2.2.3
